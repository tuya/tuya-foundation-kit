## 安装

TYFoundationKit 可以通过 [CocoaPods](https://cocoapods.org) 进行安装,   需要在 Podfile 文件中添加如下声明:

```ruby
source 'https://github.com/TuyaInc/TYPublicSpecs.git'

pod 'TYFoundationKit'
```
